PHP

Get Started

S'utilitza per interectuar en fitxers i més, es troba dins del body, un exemple:

        <?php
          echo "Mira el codi per veure aquest comentari";
        ?>

Per escriure variables en comptes de fer var i com en JS, el que fem és posar $x com per exeple:

        <?php
        $x = 5;
        echo $x;
        ?>

Hi ha multiples manreres de comentar el codi, amb #, //, /*

No distingeix entre majuscules i minuscules. Per imprimir una caracteristica en el codi el que s'ha de fer és crear la variable com hem dit anteriorment i si per exemple vols imprimr s'ha de fer: 

        echo "El meu color preferit és ". $color . ".";

Variables

Si creem una dinció dins del php, les variables fora de la funció no es poden utilitzar tan cual, les hem de declarar global: 

        $x = 5;
        $y = 10;
        function myTest() {
            global $x, $y;
            $y = $x + $y;
        }
        myTest();
        echo "<p> la variable és la $y</p>"; // outputs 15

També es pot escriure $GLOBALS['x'] per a cridar la variable

Si posem dins d'una finció static $x = 0; aquesta variable només es "declara" una vegada.
