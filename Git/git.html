Git

Getting started with Git

Clicka aqui per obrir el pdf de Git!

Git és un programari de sistema de control de versions dissenyat per Linus Torvalds, pensat en l'eficiència i confiabilitat de manteniment de versions d'aplicacions amb una enorme quantitat de fitxers de codi font. Tots els usuaris tenen una copia del code.

Configuració

git config --global user.name "Aina Palacios"
git config --global user.email "apalacios@asmws.com"
git config --list

Comandes

Help:

git help < verb >
git < verb > --help

Initialize a repository from existing cose

git init
//Ens apareix:
aina@aina-ThinkPad-E550:~/Desktop/ASM/Git/gitdemo$ git init
Initialized empty Git repository in /home/aina/Desktop/ASM/Git/gitdemo/.git/
aina@aina-ThinkPad-E550:~/Desktop/ASM/Git/gitdemo$ ls -la
total 12
drwxr-xr-x 3 aina aina 4096 jul 11 09:15 .
drwxrwxr-x 3 aina aina 4096 jul 11 08:59 ..
drwxr-xr-x 7 aina aina 4096 jul 11 09:15 .git

Per a borrar un ,git existent el que haurem de fer és aquesta comanda en el directori on està:

rm -rf .git

Add gitignore file

Utilitzes el comando següent al terminal per crear un gitignore:

touch .gotignore

I escribim aixó al .gitignore

.DS_Store
.project
*.pyc

Tenim 3 seccions, la working directory, la Starging Area i .git directory (Repository).
En el working directory és on modifiquem els files u els creem o borrem.
La staging area es on organitzem que volem commited, afegir codi retocat

Add files to staging area

git add -A//Per exemple .gitignore, -A es afegir tot
git status

Per a treure algun element committed, fem:

git reser nom.fitxer

Quan fem la comanda següen, el fitcher queda comentat i es processa, ja no es pot liminar:

git commit -m "Missatge que vols donar al que has fet"
git log //veure fitxers commited

Cloning a remote repo

git clone < url > < where to clone >
git remote -v //Info de on es troba aquests git
git branh -a //Mostra les branques (remotes i locals)

Pushing changes

Amb el copmando:

get diff

Pots veure els canvis que has fet de manera local. Un signe negatiu indica el que has tret i en un signe positiu el que has posat nou
Commitem el que hem fet amb add -Ai seguidament git commit -m "Missatge"
Ara tenim 2 comandes més,: pull i push. Apliquem la comanda:

git pull origin master

Per asegurar-nos estar al dia amb els codis i que ningú hagi fet canvis, i seguidament fem push

git push origin master

Common workflow

Create a branch for desired feature

git branch nom.branca
git checkout nom.branca

Afert commit, push branch to remote

git push -u origin nom.branca
git bramch -a

Marge a branch

git checkout master
git pull origin master
git branch --merged
git merge nom.branca
git push origin marser

Deleting a branch

git checkout master
git branch --merged
git branch -d nom.branca
git branch -a
git push origin --delete nom.branca
